YouTube video ID
YouTube mostrará un ID (como HSQE0ZCqC68), al guardar (o reproducir) un vídeo. Puede utilizar este ID y hacer referencia a su vídeo en el código HTML.

Reproducción de un vídeo de YouTube en HTML:
parareproducir el vídeo en una página web, haga lo siguiente:

  • Sube el vídeo a YouTube
  • Anote el ID de vídeo
  • Defina un elemento <iframe> en su páina web
  • Deje que el atributo src apunte a la URL del vídeo
  • Utilice los atributos width y height para especificar la dimensión del reproductor
