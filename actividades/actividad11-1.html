Entendiendo los CSS: inspirado en los cuadros

La clave para enteder cómo funciona CSS es imaginar que hay un cuadro invisible alredor de cada elemento HTML.

En la primera página, se ve una página HTML básica.

En la segunda página, se puede ver la misma página HTML, pero se ha agregado contornos a cada uno de los elementos para que pueda ver cómo los CSS tratará cada elemento como si estubiera dentro de su propio cuadro.
