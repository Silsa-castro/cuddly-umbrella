Floatings Elements - Elementos Flotantes

"La vida es como andar en Bicicleta. Para mantener el equilibrio debes seguir avanzando." - Albert Einstein

La propiedad float le permite tomar en elemento en el flujo normal y colocarlo tan lejos a la izquierda o derecha del elemento contenedor como sea posible. Cualquier cosa que se sienta dentro del elemento contenedor fluira alrededor del elemento que esta flotando.

Cuando se utiliza la propiedad float, tambien se debe utilizar la propiedad width para indicar el ancho que debe tener el elemento flotante. Si no se hace, los resultados pueden ser incoherentes, pero es probable que el cuadro tome el ancho completo del contenedor (al igual que en el flujo normal).
